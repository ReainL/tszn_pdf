Microsoft 与 Datazen

  • 微软在 2015年4月15日 收购移动商业智能 BI 和可视化领域厂商 Datazen。
  • 在未来的计划中，微软可能将 Datazen 整合到 Power BI 中以对现有的解决方案形成补充。
  • SQL Server 企业版 2008 以及以后的版本都可以免费下载 Datazen Server 软件。

关于更多收购的内容大家可以参看 www.datazen.com

Datazen 概述之跨平台的 Apps 应用 (Windows, IOS, Android)

Datazen Windows 8 App 可以基于 Excel, 云或者企业资源数据库创建和发布报表。

  • Microsoft SQL Server
  • Microsoft SQL Server Analysis Services
  • Microsoft SQL Azure
  • Microsoft SharePoint Lists
  • Excel documents in network shares
  • Generic ODBC data sources
  • Other data sources available upon request

从上面可以看出 Datazen 的数据源支持还是以微软为主，包括 Datazen Server 的部署，以及后面会介绍到的 Datazen Publisher 都是基于微软平台的，这是微软收购 Datazen 很重要的一个原因。Datazen 相当于一个外在插件可以和微软 BI 的系列产品无缝集成，丰富了微软产品线的同时也起到了保护微软产品的作用，可以看到上面的产品支持就知道了。   发布到 Datazen 服务器上之后，BI Dashboards 可以通过这些移动设备的 App 应用或者通过主要的支持 HTML5 浏览器来访问。

Datazen 几乎支持所有主流移动平台 – Windows 8 and later、Windows Phone、IOS 7 and later、Android 4.1 and later。

HTML 5 的支持 - 每一个 Datazen 企业版服务器发布的都是以 HTML 5 标准的 Dashboard。  通过登录屏幕进行身份验证，然后选择所需要查看的 Dashboard，这些权限是通过 Datazen Server 来配置完成的。  也可以集成到其它应用，比如 SharePoint。  

KPI 资源库

强大的 KPI 资源库，各种 KPI 指标图形支持。  可以基于实时数据及时发布 KPI 指标和趋势，非常优质的用户体验。  每一个 KPI 指标可以与一个或者多个 Datazen 的 Dashboard 图表关联，比如进行一些 Drill-Down 或者更深一层次的选择性的数据分析。

可视化的 BI 仪表盘 Dashboard 和数据分析

  • 统一的 Dashboard 设计器，单点设计部署，多平台访问，支持任何屏幕分辨率。
  • 简单 Simple - 基本上不需要太多的技能就可以自己动手设计报表，Excel 级别用户。
  • 灵活 Flexible – 几乎支持任何的报表图形，丰富的数据源访问支持。
  • 美观 Beautiful – Datazen Dashboard Designer makes it impossible to create visually unattractive dashboards.
  • 在设计阶段就可以预览数据以检查是否满足设计需求。
  • 可以集成多种数据源的数据到一个单独的 Dashboard 中。
  • 对于 Excel 中的数据，Cloud 云以及其它企业级的数据源处理是一样的方式。
  • 可以在本地设计时随时保存 Dashboard 并通过设计器发布到 Datazen Server 上。
  • 有能力实现各种布局方式，专业的、极具视觉冲击力的外观设计。

在移动手机端的效果，一次设计，一次部署，各个客户端格式都可以自动适应调整。

带有下钻功能，可以进行报表跳转。

对大数据集的支持

  • Datazen 的数据处理策略是只加载在报表上所需要展现的数据。
  • 用户交互数据的时候再次从服务器上加载数据 – Parameterized Server Data Views。
  • Server-based data view – 相当于在服务器上参数化查询，根据参数创建 Data Views。

多自定义地图的支持

  • Datazen 有多达 70 余种的地图集合，包括了世界上主要的国家以及地区。
  • 另外还可以自定义地图，需要基于 ESRI Shape file format http://en.wikipedia.org/wiki/Shapefile

高效的团队协作 Team Collaboration

每一个 Datazen KPI 和 Dashboard 都包括一个活动信息流 activity stream。  在这个活动信息流中包含了一些重要的事件，比如 KPI/Dashboard 的创建和更新，以及用户对 KPI/Dashboard 中的一些评论，批注等。

自定义品牌

Datazen 预留了 Logo 的位置，从 Apps 中看就像是自己品牌的一个应用。

部署与管理

  • 企业级的安全验证与数据安全控制[list][*]Active Directory authentication
  • Active Directory Federation Services (ADFS) authentication
  • Granular permissions controls for all dashboards and data views
  • Secure device-server communication through SSL
  • Data encryption at rest on Datazen Server
  • Data encryption on the client in case of device loss
  • PIN lock screens in all mobile client
  • 服务器端的数据缓存
  • 客户端的数据缓存，离线访问
  • 用户可以将报表暂时保存在本地，或者发布到服务器上，也有非常好的权限控制。

Datazen 初步体验感受

  1. 微软的这个策略是正确的，在移动端商业智能 BI 报表领域推进布局，移动 BI 的发展在未来是一个重点方向。
  2. 只要企业拥有 SQL Server 企业版 2008 和以后的版本就可以免费使用 Datazen，更加丰富了微软 BI 系列产品和解决方案 （SQL Server + SSIS + SSAS + SSRS + Power BI + Power View + Datazen）, 当然其中有些产品可能会慢慢被替代除非做到真正的产品革新，比如 SSRS、Power View。
  3. Datazen 只适合做展现，不适合做任何类型的表达式计算，格式良好，维度度量结构清晰是前提。
  4. 对于在移动端能够存储的数据量目前还未做进一步的验证，据了解不超过10W行（这一点需要进一步的确认）。总之，不要尝试在 Datazen 上浏览细节数据。
  5. Datazen Server 的部署与权限管理目前需要进一步的研究。
