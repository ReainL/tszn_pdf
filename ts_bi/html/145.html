开篇介绍

新安装了 SQL Server 2012 和 Visual Studio 2012。

准备在 VS 2012 中创建一个 BI 项目，但是发现找不到这一项。

后来想起来在装完 SQL Server 2012 以后，以前的 BIDS 工具的名称也变成了 SQL Server Data Tools (SSDT)，但是实际上使用的还是 Visual Studio 2010 Shell。

集成开发插件

查看了一下资料，原来如果要在 Visual Studio 2012 中集成 BI 项目的话还需要集成一个插件。

Microsoft SQL Server Data Tools - Business Intelligence for Visual Studio 2012

http://www.microsoft.com/en-gb/download/details.aspx?id=36843

 

下载这个插件安装完成后重启一下 Visual Studio 2012 就可以使用了，当然这也不一定要安装，也可以直接在 SSDT 中创建和开发 BI 项目即可。

另外在安装 SQL Server 2012 的时候也要注意，它有操作系统版本的安装要求，比如本人是 Win 7 操作系统，那么就必须要求是 SP1 版本的才可以安装。

Visual Studio 2012 下载地址 - http://www.microsoft.com/visualstudio/eng#downloads

SQL Server 2012 下载地址 - http://www.microsoft.com/en-us/sqlserver/get-sql-server/try-it.aspx
