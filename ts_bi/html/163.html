原以为被忽悠

今天在朋友圈里看到几个 Tableau 的朋友同时发了一张非常有意思的截图，作为微软的铁杆粉丝，以为是坏坏的Tableau 朋友在不怀好意的调戏我软，这图….肯定是PS 的。



事实是惊呆了

后来顺着地址查了一下，惊呆了！ 如图所示：报表的最下方赫然出现了 Tableau 的Logo，很明显在微软必应推广的网站中，这个有关市场推广流量图的分析是用 Tableau 来做的，而不是微软自己的 SSRS 或者 Power BI。

 

这篇文章和图表分析大意也就是说：大部分的市场推广人员或企业主都跟你们一样，在市场推广研究工具方面的预算很有限。所以应该做好成本控制，有的放矢地投放广告。要清楚知道每周哪几天访问量最大，一天中又是哪个时间段是访问高峰，再针对性地在Bing上面投放广告。例如下面这个搜索工具的表格，统计出了在Bing上面做广告的流量图，一周每天的流量变化，以及一天当中哪个时间段访问量最大。

很简单的分析

这是原本是一个很简单的分析，也是一个非常简单的数据报表展现，但是非常的遗憾，微软的这位数据分析人员并没有使用微软自己的产品，比如 Microsoft BI 体系中的SSRS报表，或者 Power BI。

微软 SSRS 报表（BIWORK 作为微软的一个小粉丝，把WIN 8 风格 Metro UI 都搬到报表中了，我对微软是真爱啊！）




微软 Power BI



要知道在Gartner 2016年2月发布的商业智能BI和分析平台中位于Leaders 领导者象限只有三家，分别是 Tableau，Qlik和 Microsoft。而在这个魔力象限图中，Microsoft 实质上指的就是 Microsoft Power BI ，而 Power BI 的竞品直指的其实就是 Tableau。


我当时写过一篇文章解析这个魔力象限，有一部分内容我也做出了说明：

“ Microsoft 在前瞻性（Completeness of Vision）方面进步很快，已经处于领导者象限前瞻性的第一集团位置，说明 Microsoft 已经对现有的商业智能和分析平台的工具做出了大量的补充和创新，投入了较大的精力。从2015年收购了 Datazen 移动端产品并且加快了 Power BI 产品的推进。”

可以点击阅读我写的这篇文章：如何看懂 Gartner 魔力象限？再度解析 Gartner 2016商业智能与分析平台魔力象限
